## 模块职责

### 1. CLI 层 (`cli/main.py`)
- 命令行入口
- 命令：`stat`, `list`, `export`, `analyze`

### 2. 核心记录器 (`recorder.py`)
- `ExceptionRecorder`: 主记录器类
- `record()`: 记录异常
- `get_stats()`: 获取统计

### 3. 分类器 (`classifiers.py`)
- `ErrorClassifier`: 错误类型自动识别
- 11 种错误类型映射
- 重试策略推荐

### 4. 存储层 (`storage.py`)
- `ErrorStorage`: 本地 JSON 存储
- 支持按时间范围查询

### 5. 报告器 (`reporters.py`)
- `CLIReporter`: 终端输出
- `MarkdownReporter`: Markdown 导出

## 数据流

```
Exception 发生
      │
      ▼
┌─────────────┐
│  record()   │ ◄─── 用户调用
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ Classifier  │ ──── 自动分类错误类型
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ ErrorRecord │ ──── 构建结构化记录
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  Storage    │ ──── 持久化到本地
└─────────────┘
```
